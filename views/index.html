O que é ANGULARJS?

  • Framework JavaScript com grande foco em HTML

    • HTML + JS

  • Desenvolvido pelo Google em 2009

    • ...originalmente por Misko Hevery e Adam Abrons

  • Arquitetura MVC

    • Podendo variar (MV*)...

  • Suporta os navegadores modernos (Desktop e Mobile)

    • EI8+, Chrome, Firefox, Safari e Opera

+

"HTML enhanced for web apps!"

  • O AngularJS funciona como uma extensão ao documento HTML.

    • Novos atributos no html para conseguir adicionar funcionalidades extras

  • AngularJS é quase uma linguagem declarativa

    • Adicionando novos parâmetros é possivel alterar o comportamento padrão do html

  • Fornece um conjunto de funcionalidades que tornam o desenvolvimento web muito mais fácil

    • DataBinding, templates...

  • Aplicações baseadas em browser

    • Sinlge-page Apps...

Porque usar ?

No contexto de uma aplicação

com alta complexidade e dinamismo da página html...

Inicialização e encapsulamento dos demais componentes descritos 

            
      
//Inicialização do módulo
var module = angular.module('AngularIntro', []);
 
//Adição do Controller ao módulo
module.controller('SimpleController', function($scope){
  $scope.someThings = ['First', 'Angular', 'Module'];
    $scope.addThing = function(){
    var thing = $scope.thing;
    if($scope.someThings.indexOf(thing) == -1)
      $scope.bairroDoLimoeiro.push(thing);

Routes

Ligação entre Controller e View (HTML partials)

            
var app = angular.module('MyApp', ['ngRoute', 'simpleController']);
            
          
            
app.config(['$routeProvider', 
  function($routeProvider) {
    $routeProvider.when('/home', {
      templateUrl: 'partials/home.html',

    });
  }
]);

Services

O que é? Para que serve?


  • Objetos Singletons que realizam tarefas específicas comuns em uma aplicação WEB
  • Compartilhamento de informações entre controller
  • Comunicação com o servidor
  • Concentrar a lógica de negócio deixando o fluxo da aplicação por conta do controller

Na prática...

            
// Service            
module.factory('simpleService', function(){
  var service = {};
  service.nameList = ['Joe', 'Lenny', 'Colt', 'Julie']; 

  service.addName = function(name){
    if(service.nameList.indexOf(name) == -1)
      service.nameList.push(name); 
  }
 
  return service;
});
            
          
            
// Consumindo no controller
